print() 函數(一)

  • object –複數，表示可以輸出多個對象，用逗號,分隔。
  • sep –間隔多個對象，默認一個空格。
  • end –結尾，默認換行符號\n。
  • file –寫入文件對象。
  • flush –輸出是否被緩存，由file決定flush=True，會被強制刷新。
